# Annotation Editor

The annotation editor provides 14 professional tools for marking up screenshots.

## Opening the Editor

- **Quick Access Overlay:** Click "Edit" or press `Cmd+E` after capture
- **Capture History:** Right-click a capture and select "Open in Editor"

## Tool Categories

### Drawing Tools
- Rectangle (Outline) - Unfilled rectangle
- Rectangle (Solid) - Filled rectangle
- Circle - Unfilled circle
- Line - Straight line
- Arrow - Directional arrow

### Markup Tools
- Text - Text labels with fonts
- Pencil - Freehand drawing
- Highlighter - Semi-transparent marking
- Numbered Step - Sequential markers

### Utility Tools
- Select - Select and move annotations
- Crop - Crop image
- Blur - Blur sensitive areas

See the [Tools Reference](/docs/annotation/tools) for detailed information on each tool.

## Quick Start

1. Open a screenshot in the editor
2. Select a tool from the toolbar
3. Choose a color from the palette
4. Draw on the image
5. Save or export when done

# Annotation Tools

ScreenCapture provides 14 tools for annotating screenshots.

## Tool Reference

| Tool | Description |
|------|-------------|
| Select | Select and move annotations |
| Crop | Crop image to selection |
| Rectangle (Outline) | Draw unfilled rectangle |
| Rectangle (Solid) | Draw filled rectangle |
| Circle | Draw unfilled circle |
| Line | Draw straight line |
| Arrow | Draw directional arrow |
| Text | Add text labels |
| Blur | Blur sensitive areas |
| Pencil | Freehand drawing |
| Highlighter | Semi-transparent highlighting |
| Numbered Step | Sequential numbered markers |

## Select Tool

Select and manipulate existing annotations.

- **Click** to select an annotation
- **Drag** to move the selection
- **Delete** to remove selected annotation

## Crop Tool

Crop the image to a specific area.

- Draw a rectangle to define the crop area
- Press Enter or click "Apply" to crop

## Rectangle

Draw rectangles in outline or solid styles.

- **Outline:** Unfilled border only
- **Solid:** Filled shape

## Circle

Draw unfilled circles and ellipses.

- Hold **Shift** to constrain to a perfect circle

## Line

Draw straight lines.

- Hold **Shift** to constrain to 45Â° angles

## Arrow

Draw directional arrows to point out elements.

- Arrow head appears at the end point

## Text

Add text labels to your screenshots with native macOS styling.

### Appearance

Text boxes feature a frosted glass effect with a native macOS appearance:

- **Background:** Translucent frosted glass material
- **Border:** Dashed border when selected for clear visibility
- **Resize Handles:** 8 handles (corners + edges) for precise resizing

### Fonts Available
- System Default
- SF Pro
- SF Mono
- Helvetica
- Times New Roman
- Georgia
- Courier

### Size Range
4pt to 100pt

### Behavior

- **Size Persistence:** Text box dimensions are preserved after deselection
- **Auto-Save:** Content saves automatically when switching to another tool
- **Editing:** Click to edit, click outside or switch tools to finish

## Blur

Blur sensitive areas like personal information.

| Setting | Value |
|---------|-------|
| Blur Radius | 10 |
| Style | Gaussian |

## Pencil

Freehand drawing for natural annotations.

- Pressure-sensitive if using trackpad
- Configurable stroke width

## Highlighter

Semi-transparent highlighting for emphasis.

- 50% opacity for visibility without obscuring
- Wide stroke width by default

## Numbered Step

Add sequential numbered markers for tutorials.

- Numbers auto-increment
- Circular markers with number inside
- Great for step-by-step guides

# Capture Area

Interactive region selection allows you to capture exactly the portion of the screen you need.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+4` |
| Native Mode | `Cmd+Shift+4` |

## How to Use

1. Press the keyboard shortcut
2. Click and drag to select the desired area
3. Release to capture

## Settings

| Setting | Value |
|---------|-------|
| Format | PNG |
| Dimensions | Shown during selection |
| Magnifier | Optional (configurable in Settings) |

## Tips

- Hold `Shift` while dragging to constrain to a square
- Press `Esc` to exit pre-capture immediately
- The selection dimensions display in real-time

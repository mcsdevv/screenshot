# Capture Fullscreen

Instantly capture the entire primary display.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+3` |
| Native Mode | `Cmd+Shift+3` |

## How to Use

1. Press the keyboard shortcut
2. The entire screen is captured immediately

## Settings

| Setting | Value |
|---------|-------|
| Format | Configured screenshot format (PNG/JPEG/TIFF) |
| Display | Primary display |
| Cursor | Configurable (include or exclude) |

## Options

### Include Cursor

Choose whether to include the mouse cursor in the capture.

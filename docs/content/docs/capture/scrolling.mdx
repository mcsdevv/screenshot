# Scrolling Capture

Capture long content like web pages or documents by automatically stitching multiple frames together.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+6` |
| Native Mode | `Cmd+Shift+6` |

## How to Use

1. Press the keyboard shortcut
2. Select the scrollable area
3. Let the app automatically scroll and capture
4. Images are stitched together seamlessly

## Settings

| Setting | Value |
|---------|-------|
| Format | PNG (stitched) |
| Resolution | 2x (Retina) |
| Overlap Detection | 10% |

## Technical Details

- **Frame Overlap:** 10% overlap between frames for seamless stitching
- **Resolution:** Captures at 2x Retina resolution for sharp output
- **Algorithm:** Intelligent frame alignment and blending

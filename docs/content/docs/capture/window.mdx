# Capture Window

Capture a specific window with optional shadow effects.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+5` |
| Native Mode | `Cmd+Shift+5` |

## How to Use

1. Press the keyboard shortcut
2. Move cursor over the desired window
3. Click to capture

## Settings

| Setting | Value |
|---------|-------|
| Format | PNG |
| Shadow | Optional (configurable) |
| Rounded Corners | Preserved |

## Options

### Include Window Shadow

When enabled, captures the native macOS window shadow for a more polished look.

### Rounded Corners

Window corners are preserved in their native rounded state.

## Tips

- Press `Esc` to exit pre-capture immediately

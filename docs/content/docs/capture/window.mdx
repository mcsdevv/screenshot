# Capture Window

Capture a specific window with the native picker.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+5` |
| Native Mode | `Cmd+Shift+5` |

## How to Use

1. Press the keyboard shortcut
2. Move cursor over the desired window
3. Click to capture

## Settings

| Setting | Value |
|---------|-------|
| Format | Configured screenshot format (PNG/JPEG/TIFF) |
| Shadow | Not included in native quick capture mode |
| Rounded Corners | Preserved where available from source window |

## Tips

- Press `Esc` to exit pre-capture immediately

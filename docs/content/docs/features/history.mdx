# Capture History

Browse, search, and manage all your captures in one place.

## Opening History

- **Keyboard Shortcut:** `Cmd+Shift+H`
- **Menu Bar:** Click icon > Show Capture History

## Views

### Grid View

| Setting | Value |
|---------|-------|
| Layout | 2-column adaptive |
| Card Size | 220-280px |
| Thumbnails | Auto-generated |

### List View

Compact rows with small thumbnails for quick scanning.

## Filtering

Filter captures by type:

| Filter | Shows |
|--------|-------|
| All | All capture types |
| Screenshot | Static screenshots |
| Scrolling | Scrolling captures |
| Recording | Video recordings |
| GIF | Animated GIFs |

Search by filename for quick access.

## Sorting

| Sort Order | Description |
|------------|-------------|
| Date (Newest) | Most recent first |
| Date (Oldest) | Oldest first |
| Name (A-Z) | Alphabetical |
| Name (Z-A) | Reverse alphabetical |

## Per-Capture Actions

Right-click any capture for options:

| Action | Description |
|--------|-------------|
| Open | Open in default app |
| Open in Editor | Open in annotation editor |
| Copy to Clipboard | Copy image |
| Save As... | Export to new location |
| Show in Finder | Reveal file |
| Toggle Favorite | Mark/unmark as favorite |
| Delete | Remove capture |

## Auto-Cleanup

Automatically remove old captures to save space.

| Setting | Options |
|---------|---------|
| Retention Period | 7, 14, 30, or 90 days |
| Favorites | Never auto-deleted |
| Default | 30 days |

Configure auto-cleanup in [Settings > Storage](/docs/settings).

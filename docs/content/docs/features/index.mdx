# Features

Beyond capture and annotation, ScreenCapture offers powerful productivity features.

## Feature Overview

| Feature | Description |
|---------|-------------|
| [Quick Access](/docs/features/quick-access) | Post-capture action overlay |
| [OCR](/docs/features/ocr) | Text extraction in 10+ languages |
| [Pinned Screenshots](/docs/features/pinned) | Floating reference windows |
| [Webcam Overlay](/docs/features/webcam) | Camera feed during capture |
| [Capture History](/docs/features/history) | Browse and manage captures |

## Quick Links

- **Quick Access Overlay** - Appears after every capture with common actions
- **Text Extraction** - Extract text from screenshots instantly
- **Pinned Screenshots** - Keep references always visible
- **Webcam Overlay** - Include your face in recordings
- **Capture History** - Find and manage all your captures

# Text Extraction (OCR)

Extract text from screenshots using Apple's Vision framework.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+O` |
| Native Mode | `Cmd+Shift+O` |

## Supported Languages

| Language | Code |
|----------|------|
| English (US) | en-US |
| English (GB) | en-GB |
| German | de |
| French | fr |
| Spanish | es |
| Italian | it |
| Portuguese (Brazil) | pt-BR |
| Simplified Chinese | zh-Hans |
| Traditional Chinese | zh-Hant |
| Japanese | ja |
| Korean | ko |

## Features

| Feature | Description |
|---------|-------------|
| Recognition Level | Accurate |
| Language Correction | Enabled |
| Bounding Box Detection | Supported |
| Barcode Detection | Supported |

## How It Works

1. Capture or select a screenshot
2. Press the OCR shortcut or use Quick Access
3. Text is recognized and analyzed
4. Results are copied to clipboard automatically

## Use Cases

- **Extract text from images** - Copy text from graphics
- **Read receipts** - Extract amounts and items
- **Document digitization** - Convert photos to text
- **Translate content** - Extract text for translation

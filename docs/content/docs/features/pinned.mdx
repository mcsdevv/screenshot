# Pinned Screenshots

Create floating reference windows that stay visible on top of other apps.

## Creating a Pinned Screenshot

1. After capturing, click **Pin** in Quick Access overlay
2. Or press `Cmd+P` after capture
3. Or use `Ctrl+Shift+P` / `Cmd+Shift+P` globally

## Toolbar

A compact toolbar appears on hover over the pinned window, providing quick access to all controls in a single row.

### Toolbar Controls

| Control | Shortcut | Description |
|---------|----------|-------------|
| Zoom In | `Cmd++` | Increase scale (up to 3.0x) |
| Zoom Out | `Cmd+-` | Decrease scale (down to 0.5x) |
| Lock/Unlock | `Cmd+L` | Toggle interaction lock |
| Opacity | - | Cycle through opacity levels |
| Copy | `Cmd+C` | Copy image to clipboard |
| Close | `Cmd+W` | Dismiss the pinned window |

### Toolbar Layout

The toolbar uses a compact single-row layout that:
- Appears on mouse hover
- Shows tooltips with keyboard shortcuts
- Auto-hides when mouse leaves the window area

## Features

| Feature | Description |
|---------|-------------|
| Pinch-to-zoom | Trackpad gesture support |
| Draggable | Reposition anywhere |
| All Spaces | Works across all macOS Spaces |
| Fullscreen Apps | Visible over fullscreen apps |
| Multiple Windows | Unlimited simultaneous pins |

## Use Cases

- **Reference material** - Keep documentation visible
- **Design comparison** - Compare designs side-by-side
- **Tutorial following** - Keep instructions visible
- **Code snippets** - Quick reference while coding

# Getting Started

ScreenCapture is a comprehensive macOS screenshot and screen recording application with annotation tools, capture history, and system-level integration.

## Platform Requirements

- **macOS:** 14.0 or later
- **Architecture:** SwiftUI + AppKit with ScreenCaptureKit

## Quick Start

1. **Install** the application from the Mac App Store or download directly
2. **Grant permissions** when prompted (Screen Recording, Camera, Microphone)
3. **Use keyboard shortcuts** to capture:
   - `Ctrl+Shift+3` - Fullscreen capture
   - `Ctrl+Shift+4` - Area selection
   - `Ctrl+Shift+5` - Window capture

## Keyboard Shortcut Modes

ScreenCapture offers two shortcut modes:

| Mode | Modifier | Purpose |
|------|----------|---------|
| **Safe Mode** (default) | `Ctrl+Shift` | Avoids conflicts with macOS Screenshot.app |
| **Native Mode** | `Cmd+Shift` | Uses standard macOS shortcuts |

See the full [Keyboard Shortcuts](/docs/shortcuts) reference.

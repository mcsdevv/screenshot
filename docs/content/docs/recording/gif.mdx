# GIF Recording

Create animated GIFs for quick sharing and documentation.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+8` |
| Native Mode | `Cmd+Shift+8` |

## GIF Specifications

| Setting | Value |
|---------|-------|
| Format | Animated GIF |
| Frame Rate | 10, 15, or 20 FPS |
| Loop | Infinite |
| Export Pipeline | Temporary MP4 capture + `ffmpeg` GIF export |

## Quality Presets

| Preset | Max Dimension | Frame Skip | Best For |
|--------|---------------|------------|----------|
| Low | 640px | 3 | Small file size |
| Medium | 960px | 2 | Balanced quality |
| High | 1280px | 1 | High quality |

## Tips

- **Low preset** - Best for chat apps and quick sharing
- **Medium preset** - Good balance for documentation
- **High preset** - Detailed tutorials and demos
- GIF export requires `ffmpeg` to be installed

## Frame Rate

| FPS | Use Case |
|-----|----------|
| 10 | Smaller files, simple animations |
| 15 | Balanced smoothness |
| 20 | Smoother motion, larger files |

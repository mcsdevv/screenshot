# Screen Recording

Record your screen as high-quality video or animated GIFs.

## Recording Modes

| Mode | Shortcut (Safe) | Shortcut (Native) | Output |
|------|-----------------|-------------------|--------|
| [Video](/docs/recording/video) | `Ctrl+Shift+7` | `Cmd+Shift+7` | MP4 |
| [GIF](/docs/recording/gif) | `Ctrl+Shift+8` | `Cmd+Shift+8` | Animated GIF |

## Recording Controls

During recording, you have access to:
- **Pause/Resume** - Temporarily stop recording
- **Stop** - End the recording session
- **Duration Timer** - Track recording length
- **Menu Bar Indicator** - Red circle shows active recording

## Audio Options

- **Microphone** - Record voice commentary
- **System Audio** - Capture app sounds

## Cursor Options

- **Show Cursor** - Display mouse movements
- **Click Highlighting** - Highlight mouse clicks
- **Keystroke Display** - Show keyboard input

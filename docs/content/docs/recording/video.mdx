# Video Recording

Record your screen as high-quality MP4 video.

## Keyboard Shortcut

| Mode | Shortcut |
|------|----------|
| Safe Mode | `Ctrl+Shift+7` |
| Native Mode | `Cmd+Shift+7` |

## Video Specifications

| Setting | Value |
|---------|-------|
| Format | MP4 |
| Video Codec | H.264 (High Profile) |
| Video Bitrate | 10 Mbps |
| Frame Rate | 30 or 60 FPS (configurable) |

## Audio Specifications

| Setting | Value |
|---------|-------|
| Audio Codec | AAC |
| Audio Bitrate | 128 kbps |
| Sample Rate | 48 kHz stereo |

## Recording Controls

- **Stop** - End recording with duration timer
- **Cursor Display** - Optional mouse cursor visibility
- **Audio Sources** - Microphone and/or system audio

## Quality Presets

| Preset | Resolution | Frame Rate |
|--------|------------|------------|
| Low | 720p | 30 FPS |
| Medium | 1080p | 30 FPS |
| High | Native | 60 FPS |

# Settings

Configure ScreenCapture through the Preferences window.

**Open Settings:** `Cmd+,` or Menu Bar > Preferences

## General

| Setting | Description |
|---------|-------------|
| Launch at login | Start ScreenCapture on system boot |
| Show menu bar icon | Display camera icon in menu bar |
| Play sound after capture | Audio feedback on capture |
| Show Quick Access overlay | Display post-capture actions |
| Quick Access auto-dismiss | Auto-hide timing (3s/5s/10s/Never) |
| Popup position | Corner used for Quick Access and pinned screenshots |
| Default action after capture | What happens automatically |

## Shortcuts

| Setting | Description |
|---------|-------------|
| Shortcut Mode | Safe Mode (`Ctrl+Shift`) or Native Mode (`Cmd+Shift`) |
| Current shortcuts | View active key combinations |
| System shortcut status | Integration with macOS shortcuts |

## Capture

| Setting | Description |
|---------|-------------|
| Include cursor | Show mouse in screenshots |
| Image format | PNG, JPEG, or TIFF |
| JPEG quality | 10-100% (if JPEG selected) |

## Recording

| Setting | Description |
|---------|-------------|
| Video quality | Low, Medium, or High |
| Frame rate | 30 or 60 FPS |
| Show cursor | Include cursor in recording |
| Microphone recording | Capture voice |
| System audio recording | Capture app sounds |
| Mouse click highlighting | Highlight clicks |
| Keystroke overlay | Displayed as unavailable in native capture mode |

## Storage

| Setting | Description |
|---------|-------------|
| Save location | Default, Desktop, or Custom |
| Storage usage | Current disk usage |
| Auto-cleanup | Enable automatic deletion |
| Cleanup retention | 7, 14, 30, or 90 days |
| Clear all captures | Delete all history |
| Open folder | Reveal storage location |

## Advanced

| Setting | Description |
|---------|-------------|
| Debug log file | Open the current log file |
| Log directory | Open the logs folder |
| Reset preferences | Restore defaults |
| Version | Current app version and build |

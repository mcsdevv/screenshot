# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in ScreenCapture.

## Shortcut Modes

ScreenCapture offers two shortcut modes to avoid conflicts with macOS:

| Mode | Modifier | Description |
|------|----------|-------------|
| **Safe Mode** (default) | `Ctrl+Shift` | Avoids conflicts with macOS Screenshot.app |
| **Native Mode** | `Cmd+Shift` | Uses standard macOS shortcuts |

Switch modes in [Settings > Shortcuts](/docs/settings).

## Native Mode Configuration

When using Native Mode, you may need to disable or remap the built-in macOS screenshot shortcuts:

1. Open **System Settings > Keyboard > Keyboard Shortcuts**
2. Select **Screenshots** from the sidebar
3. Uncheck or modify the conflicting shortcuts
